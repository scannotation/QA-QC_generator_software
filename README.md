The QA/QC generator evaluates the completeness and robustness of MS1 HRMS data preprocessing.
This tool produces harmonized PARC QA/QC preprocessing outputs and automatically generates a report based on the HBM4EU analytical QA/QC framework and additional project-specific criteria. (for further details, please refer to: https://doi.org/10.1016/j.trac.2024.117674).\
\
For more information, view the video tutorial by downloading the file ["QA-QC generator-tutorial.zip"](https://github.com/scannotation/QA-QC_generator_software/blob/main/QA-QC_generator-tutorial.zip).

## Development
QA-QC generator was developed in Python 3.9 on Windows 10.
